Projection for Linear Constraint

project x such that Ax=b. January 5, 2017

Consider the problem: project x, such that P(x) satisfies A(P(x))=b.

First, we need to figure out the definition of projection here: find y such that y=argminz ||x-z||2, s.t. Az=b.

The set {z:Az=b}=set {z:z=v+u,Au=0}, where v is a given solution satisfying Av=b.

Also, u satisfies Au=0 ↔ u=p-A'(AA')-1Ap, p ∈ R.

Therefore, the problem reduces to find q=argminp||x-(v+p-A'(AA')-1Ap)||2, which can be easily solved by taking the first derivative of object function.
