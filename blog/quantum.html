Complex and 2-norm generalization on probability

The classic probability system is based on non-negative and 1-norm, which comes naturally from the classical probability model. Quantum system generalizes it by describing a qubit's "random" state with complex system and 2-norm probability:
For one qubit, (α,β)→α|0>+β|1>, |α|2+|β|2=1. The probability of seeing 0 is |α|2, and the probability of seeing 1 is |β|2=1. Therefore, the set of all status for one qubit forms a circle.

Operation: unitary matrix, transform qubit from one state to another state

The most natural 2norm-preservation operation.

Mixed States

Given a mixed state, calculate density matrix according to the distribution on different states. If two distribution gives the same density matrix, then these two are indistinguable, that is, the same mixed state. To describe a N-dimensional mixed state, we need N2 independent real parameters to describe the Hermitian density matrix.

Why 2-norm?

  • Only in p≤2 cases, can we get the non-trivial linear transformations that preserve p-norm.
  • cute litte fact: xp+yp=zp. Only in p≤2 cases, can we get nontrivial integer solution.

    • Why Complex System

  • A mathematical observation: Comlex numbers are algebraically closed.
  • Continuity assumption: any linear transformation is continuous: for any U, there exists V s.t. V2=U;
  • Freedom coherence for combining mixed states: Use tensor product to combine two quantum system states. Therefore, for one quantum system A and another quantum system B, we use (NANB)2=NA2NB2 independent real parameters to describe the combined mixed states.
  • Uniform Random Distribution: choose a quantum state uniformly at random, then the resulting probability vector is also distributed uniformly at random.

    No-Cloning Theorem

    Theory: There is no prodecure, consistent with quantum mechanics, that takes an unknown quantum state as input, and two separate copies as output.

  • its connection with Uncertainty Principle
  • The famous application: Quantum Key Distribution(QKD), which can only be eavesdropped by the "man(woman)-in-the-middle" attack.
